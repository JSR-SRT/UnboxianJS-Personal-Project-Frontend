# Personal Project

## Overview

A responsive e-commerce web application for art toy collectibles, built with React and Tailwind CSS.
It provides a smooth and modern shopping experience with features like product browsing, toy image galleries, cart management, and order tracking.

The frontend is designed with a scalable architecture and is fully compatible with the backend built using Node.js, Express.js, and MongoDB.
Future enhancements include payment integration and inventory management.

## Tech Stack

React — Component-based UI

Vite — Fast development environment

Tailwind CSS — Utility-first styling

React Router — Client-side routing

Shadcn/UI — Modern component library

## Features

Responsive UI across devices

User Login

Dynamic product listing and image gallery

Add to cart and order confirmation flow
